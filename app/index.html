About the site

Thank you for taking the time to visit my site. This is just a temporary site until I get my VPS setup so I can start serving pages from a Node/Express API or a combination of Express and Ruby on Rails. The site will showcase my skills as a frontend developer and a linux server administrator. I am using a build script that I created with Grunt.js that sets up frontend projects with SaSS and a folder structure that lends itself for agency work.

My main site will be an isomorphic JavaScript app that will consume a JSON api either from Express or RoR, or a combination of the two, depending on the route I take. The frontend will use the React.js library to generate the views after the initial download from the server, react will take over rendering content from the API, hence, isomorphic web app. I will use Digital Ocean to host the site. It's a great resource and at a fair price.

This site is using the Skrollr JS library for creating parallax and scrolling animations. It's mostly for eye candy but also lends itself to the overall UX for the visitor. I can only host HTML/CSS/JS sites that only rely on Apache and PHP because of the current hosting situation. Most of the work is responsive but there might be some older projects I did before RWD became main stream. I hope you enjoy what you see and thank you once again for stopping by.
